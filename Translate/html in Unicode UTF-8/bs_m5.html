Для создания свитка заклинаний потребуется чистый свиток необходимого уровня, а также денежный взнос.
Вот список заклинаний, которые можно запечатать в свитке:

[Заклинания 1 уровня]
стоимость: 50 аден
Свет / Щит / Стрела энергии / Святое оружие / Малое лечение
[Заклинания 2 уровня]
стоимость: 100 аден
Уменьшить вес / Ответное обнаружение / Зачаровать оружие / Лечить отравление / Огненная стрела
[Заклинания 3 уровня]
стоимость: 100 аден
Молния / Благословить доспех / Лечение / Облако мороза / Испугать нежить
[Заклинания 4 уровня]
стоимость: 200 аден
Медитация / Огненный шар / Чары ловкости / Счётчик магии / Замедление
[Заклинания 5 уровня]
стоимость: 200 аден
Великое лечение / Снять проклятие / Кража маны / Стрела холода / Призвать молнии

Для большинства заклинаний требуется только чистый свиток и плата за создание. Но для таких заклинаний, как "Счётчик магии" и "Замедление", требуются дополнительно магические самоцветы.
