Хранитель склада:

Установка пароля хранилища
 1. Выберите пункт "Установить/Изменить пароль".
2. Выберите "OK".
3. Введите новый пароль, за тем кликните "OK". (Пароль должен быть 6-значным числом.)
4. Введите новый пароль ещё раз и кликните "ОК" для подтверждения выбранного пароля.
5. Пароль хранилища установлен.

Использование хранилища
 1. Выберите пункт "Забрать предмет".
2. Введите шестизначный пароль, который вы установили ранее и нажмите "OK".
3. Заберите необходимые вам предметы из хранилища.
4. Вы также можете использовать свой пароль личного хранилища при получении предметов из хранилища клана.

Изменение пароля хранилища
 1. Выберите пункт "Установить/Изменить пароль".
2. Введите текущий пароль, затем нажмите "OK".
3. Введите новый пароль, затем нажмите "OK". (Пароль должен быть 6-значным числом.)
4. Введите новый пароль ещё раз и кликните "ОК" для подтверждения выбранного пароля.
5. Пароль хранилища изменён.

Удаления пароля хранилища
 1. Выберите пункт "Установить/Изменить пароль".
2. Введите текущий пароль, затем нажмите "OK".
3. Нажмите "OK", оставляя поле для нового пароля пустым.
4. Нажмите "OK", оставляя поле для подтверждения нового пароля пустым.
5. Пароль хранилища удалён.

Примечание
 1. Пароль хранилища должен быть 6-значным числом.
2. Вы можете установить пароль хранилища в меню Настройки игры.
